Supervised Learning

Hover to flip →
Definition

A learning paradigm where a model trains on labeled data — pairs of inputs and known correct outputs — to learn a prediction function.

Concept

Overfitting

Hover to flip →
Definition

When a model learns training data too well — including noise — so it performs poorly on new, unseen data. High training accuracy, low test accuracy.

Algorithm

K-Means

Hover to flip →
Definition

An unsupervised clustering algorithm that groups data into K clusters by iteratively assigning points to the nearest centroid and updating centroids.

Architecture

Neural Network

Hover to flip →
Definition

A model with layers of interconnected nodes inspired by biological neurons. Each node applies a weighted sum and activation function to its inputs.

Concept

Gradient Descent

Hover to flip →
Definition

An optimization algorithm that adjusts model parameters in the direction of steepest loss decrease to minimize prediction error during training.

Algorithm

SVM

Hover to flip →
Definition

Support Vector Machine — finds the hyperplane that maximizes the margin between two classes. The nearest training points to the hyperplane are "support vectors."

Paradigm

Reinforcement Learning

Hover to flip →
Definition

A learning paradigm where an agent learns by interacting with an environment, receiving rewards or penalties for actions, and aiming to maximize long-term reward.

Concept

Bias–Variance Tradeoff

Hover to flip →
Definition

Bias is error from oversimplified models (underfitting). Variance is error from over-complex models (overfitting). Good ML models balance both for optimal generalization.

Method

Backpropagation

Hover to flip →
Definition

Algorithm for training neural networks: computes the gradient of the loss with respect to each weight by propagating errors backwards through the network.
