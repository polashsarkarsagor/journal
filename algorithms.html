Linear Regression

Supervised · Regression
▼

Linear regression models the relationship between a dependent variable and one or more independent variables by fitting a straight line (or hyperplane) through the data. The goal is to find the line that minimizes the sum of squared differences between predictions and actual values — called the "least squares" method.

The model is expressed as: ŷ = β₀ + β₁x₁ + β₂x₂ + ... + ε, where β values are learned coefficients and ε is error.

House price prediction Sales forecasting Risk assessment Trend analysis
🔀

Logistic Regression

Supervised · Classification
▼

Despite its name, logistic regression is a classification algorithm. It uses the logistic (sigmoid) function to squash a linear combination of inputs into a value between 0 and 1, which is interpreted as a probability. If the probability exceeds 0.5, the input is classified as class 1; otherwise class 0.

The sigmoid function is: σ(z) = 1 / (1 + e⁻ᶻ). It works best when the relationship between features and the target is approximately log-linear.

Spam detection Disease diagnosis Credit scoring Sentiment analysis
🌳

Decision Trees

Supervised · Classification & Regression
▼

Decision Trees partition the feature space by asking a series of yes/no questions. At each internal node, the tree splits on the feature and threshold that best separates the data (maximizing information gain or minimizing Gini impurity). The process continues recursively until a stopping criterion is met.

They are highly interpretable — you can trace exactly why a prediction was made by following the path from root to leaf. However, single trees are prone to overfitting on complex data.

Medical diagnosis Customer segmentation Fraud detection Feature selection
🌲

Random Forests

Ensemble · Classification & Regression
▼

A Random Forest builds many decision trees during training, each on a random bootstrap sample of the data and a random subset of features. At prediction time, it aggregates all trees by majority vote (classification) or averaging (regression). This technique is called "bagging."

By averaging many uncorrelated trees, Random Forests dramatically reduce variance compared to a single tree, while maintaining low bias. They're robust, resistant to overfitting, and naturally handle missing data.

Bioinformatics Stock market prediction Image classification Remote sensing
📍

K-Nearest Neighbors (KNN)

Supervised · Classification & Regression
▼

KNN is a lazy learning algorithm — it stores all training data and makes predictions only at query time. Given a new point, it finds the K most similar (nearest) training examples using a distance metric (usually Euclidean distance), then predicts by majority vote or averaging their labels.

Choosing K is critical: small K is noisy and overfits; large K is smooth but may underfit. KNN is intuitive and requires no training phase, but can be slow at inference time for large datasets.

Recommendation systems Anomaly detection Image recognition Medical diagnosis
⭕

K-Means Clustering

Unsupervised · Clustering
▼

K-Means partitions N data points into K clusters. It initializes K centroids, assigns each point to its nearest centroid, then recalculates centroids as the mean of all assigned points. This process repeats until centroids no longer move significantly.

The algorithm minimizes within-cluster sum of squares (inertia). The main challenge is choosing K — the "elbow method" plots inertia vs K values and looks for the point of diminishing returns.

Customer segmentation Document clustering Image compression Data summarization
✂️

Support Vector Machines (SVM)

Supervised · Classification & Regression
▼

SVMs find the optimal hyperplane that separates two classes with the maximum margin. The data points closest to the hyperplane are called "support vectors" — they define the margin boundary. A wider margin generally means better generalization to unseen data.

The "kernel trick" allows SVMs to classify non-linearly separable data by implicitly mapping it into a higher-dimensional space where a linear separator exists. Common kernels include RBF (Gaussian) and polynomial kernels.

Text classification Bioinformatics Face detection Handwriting recognition
🧠

Neural Networks & Deep Learning

Supervised · Universal Function Approximators
▼

Inspired by biological neurons, artificial neural networks are composed of layers of interconnected nodes. Each node computes a weighted sum of its inputs, applies a non-linear activation function (ReLU, sigmoid, tanh), and passes the result to the next layer.

Deep Learning uses networks with many hidden layers. Through backpropagation and gradient descent, the network learns optimal weights by propagating prediction error backwards through each layer. Deep networks can learn hierarchical representations — from edges, to shapes, to objects in image recognition.

Types: CNNs (images), RNNs/LSTMs (sequences), Transformers (language), GANs (generation).

Image classification Language translation Speech recognition Generative AI
