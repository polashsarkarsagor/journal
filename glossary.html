Glossary of AI & ML Terms

Quick definitions for the most important concepts in artificial intelligence and machine learning.

Activation Function

A non-linear function applied to a neuron's output, enabling neural networks to learn complex patterns. Examples: ReLU, sigmoid, tanh.

Algorithm

A step-by-step procedure for solving a problem or achieving a goal. In ML, algorithms learn patterns from data to make predictions or decisions.

Artificial Neural Network (ANN)

A computational model inspired by biological neural networks, consisting of interconnected layers of nodes that process information through weighted connections.

Backpropagation

The algorithm used to train neural networks by computing gradients of the loss function with respect to each weight and updating weights via gradient descent.

Bagging

Bootstrap Aggregating — an ensemble technique that trains multiple models on random subsets of training data and aggregates their predictions. Used in Random Forests.

Bias (Model)

Error from oversimplified assumptions in a model. High bias leads to underfitting — the model fails to capture true patterns in the data.

Batch Size

The number of training samples processed before the model's weights are updated. Larger batches are more stable; smaller batches introduce useful noise.

Classification

A supervised learning task where the model predicts which discrete category an input belongs to. Examples: spam/not spam, cat/dog, disease/no disease.

Clustering

An unsupervised learning task that groups similar data points together. Unlike classification, clusters are not predefined — the algorithm discovers them from data structure.

CNN (Convolutional Neural Network)

A deep learning architecture designed for grid-like data (images). Uses convolutional layers to learn spatial hierarchies of features: edges → textures → objects.

Cross-Validation

A technique for evaluating model generalization by splitting data into multiple train/test folds and averaging performance metrics across all folds.

Decision Tree

A tree-structured model that makes predictions by splitting data based on feature conditions. Each internal node is a decision; each leaf is a prediction.

Deep Learning

A subset of machine learning using neural networks with many hidden layers. Deep networks can learn hierarchical representations from raw data like images and text.

Dimensionality Reduction

Techniques that reduce the number of features while preserving important structure. Examples: PCA (Principal Component Analysis), t-SNE, UMAP.

Dropout

A regularization technique that randomly disables neurons during training to prevent overfitting and improve generalization in neural networks.

Embedding

A learned representation of data (words, items, users) as dense vectors in a continuous space where semantic similarity corresponds to geometric closeness.

Epoch

One complete pass through the full training dataset. Models typically train for many epochs, progressively minimizing the loss function with each pass.

Feature

An individual measurable property used as input to a model. Feature engineering — selecting and transforming features — is critical to model performance.

Feature Engineering

The process of selecting, transforming, or creating new input variables to improve a model's predictive power. Often the most impactful step in the ML pipeline.

GAN (Generative Adversarial Network)

Two neural networks — a generator and discriminator — trained adversarially. The generator creates fake data; the discriminator distinguishes real from fake. Both improve over time.

Gradient Descent

An optimization algorithm that iteratively adjusts model parameters in the direction that minimizes the loss function, analogous to rolling downhill to find a valley.

Hyperparameter

A configuration external to the model that is set before training and not learned from data. Examples: learning rate, number of layers, K in K-Means.

Inference

Using a trained model to make predictions on new data. Distinct from training — inference uses the fixed learned parameters to produce outputs for unseen inputs.

K-Nearest Neighbors (KNN)

A lazy learning algorithm that classifies new points based on the majority label of their K closest training examples, measured by a distance metric.

Label

The target output value in supervised learning. During training, labels are provided. The model learns to predict them from input features for unseen examples.

Learning Rate

A hyperparameter controlling how much model weights are adjusted per gradient descent step. Too high: training is unstable. Too low: training is slow.

Linear Regression

A supervised algorithm modeling the linear relationship between input features and a continuous output. Finds coefficients that minimize sum of squared residuals.

Logistic Regression

A classification algorithm using the sigmoid function to output probabilities for binary or multiclass outcomes. Despite its name, it is a classifier, not a regressor.

Loss Function

A mathematical measure of the difference between model predictions and actual values. Training minimizes this function. Examples: MSE for regression, cross-entropy for classification.

LSTM (Long Short-Term Memory)

A type of recurrent neural network with gates that control information flow, enabling it to learn long-range dependencies in sequential data like text and time series.

Model

A mathematical function learned from data that maps inputs to outputs. Models are defined by their architecture (structure) and parameters (learned values).

Normalization

Scaling input features to a standard range (e.g., [0,1] or zero mean). Helps gradient descent converge faster and prevents features with large values from dominating.

Overfitting

When a model performs very well on training data but poorly on test data, having learned noise and specific training examples rather than general patterns.

Precision & Recall

Precision: fraction of positive predictions that are correct. Recall: fraction of actual positives correctly identified. Both matter in imbalanced classification problems.

Random Forest

An ensemble of decision trees trained on random data subsets and feature subsets. Predictions are aggregated to reduce variance and improve generalization.

Regularization

Techniques that penalize model complexity to prevent overfitting. L1 regularization (Lasso) promotes sparsity; L2 (Ridge) penalizes large weights.

Reinforcement Learning

An ML paradigm where an agent learns by taking actions in an environment, receiving reward signals, and updating a policy to maximize cumulative reward.

Regression

A supervised learning task predicting a continuous numerical value. Examples: predicting house prices, temperature, stock values.

ReLU (Rectified Linear Unit)

Activation function defined as f(x) = max(0, x). Simple, computationally efficient, and widely used in deep learning. Avoids the vanishing gradient problem.

RNN (Recurrent Neural Network)

A neural network with connections that form cycles, allowing information to persist across time steps. Suited for sequential data like text, speech, and time series.

Support Vector Machine (SVM)

A classifier finding the maximum-margin hyperplane separating two classes. Uses kernel tricks for non-linearly separable data. Strong theoretical guarantees.

Supervised Learning

An ML paradigm where the model learns from labeled training data — input-output pairs — to predict outputs for new unseen inputs.

Training Data

The dataset used to fit model parameters. Its size and quality directly determine how well a model generalizes to new, unseen examples.

Transformer

An architecture using self-attention mechanisms to process sequences in parallel. The foundation of modern large language models (BERT, GPT, LLaMA).

Underfitting

When a model is too simple to capture underlying data patterns. Both training and test performance are poor. Caused by too few features, parameters, or training time.

Unsupervised Learning

An ML paradigm where the model learns from unlabeled data, discovering hidden patterns, structures, and groupings without predefined answers.

Variance (Model)

Error from a model's sensitivity to fluctuations in training data. High variance causes overfitting — the model learns noise and doesn't generalize.

Weight

Learnable parameters in a model (especially neural networks) that are adjusted during training. Weights determine how much influence each input has on the output.
